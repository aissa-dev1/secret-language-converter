Language Converter Help Page

English Language to Secret

To convert from English Language to Secret, use the following rules:

  • a converts to 0
  • j converts to 9
  • And so on, same thing to others...
  • k converts to ^10
  • l converts to ^11
  • And so on, same thing to others...
  • z converts to ^25

Secret Language to English

To convert from Secret Language to English, use the following rules:

  • 0 converts to a
  • 9 converts to j
  • And so on, same thing to others...
  • ^10 converts to k
  • ^11 converts to l
  • And so on, same thing to others...
  • ^25 converts to z
